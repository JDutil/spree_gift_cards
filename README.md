SpreeGiftCards
==============


Installation
=======

1. Add `spree_store_credits` and `spree_gift_cards` to Gemfile
1. Run `rails g spree_gift_cards:install`
1. Run `rails g spree_gift_cards:seed`


To run specs
=======

1. rake test_app
1. rake spec
